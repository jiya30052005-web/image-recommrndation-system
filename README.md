# 🔍 Image-Based Recommendation System

An AI-powered application that recommends visually similar images using Deep Learning.

## 🧠 Technology Used
- Python
- ResNet50 (Deep Learning Model)
- PyTorch
- Cosine Similarity
- Streamlit
- Docker-ready

## 🚀 How to Run

### Step 1 - Clone the project
git clone https://github.com/jiya30052005-web/image-recommrndation-system.git
cd image-recommrndation-system

### Step 2 - Install libraries
pip install torch torchvision streamlit pillow scikit-learn numpy requests

### Step 3 - Setup dataset
python setup.py

### Step 4 - Run the app
python -m streamlit run app.py

### Step 5 - Open browser
Go to: http://localhost:8501
Upload any image and get Top 3 similar recommendations!
